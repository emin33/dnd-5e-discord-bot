# D&D 5e Discord Bot

An LLM-powered Dungeon Master bot for Discord using the D&D 5e SRD.

## Setup

1. Copy `.env.example` to `.env` and add your Discord bot token
2. Install dependencies: `pip install -e .`
3. Run: `python -m dnd_bot.main`

## Requirements

- Python 3.11+
- Ollama with a model (default: qwen3:30b)
- Discord bot token
